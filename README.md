# 信仰游戏排行榜

AstrBot 群聊双积分排名插件，登神之路 + 觐见之梯双榜展示，支持弃誓/立誓系统、职业信仰体系、图片输出、批量录入、定时推送、全局白名单权限管控。

## 功能特性

- **双积分体系** — 登神之路 + 觐见之梯，双榜独立排名
- **职业信仰** — 6 职业 x 6 信仰，角色定制体系
- **弃誓系统** — 弃誓标记永久保留，排行榜显示(弃誓者)，可配置弃誓文本
- **立誓/设置职业** — 信仰与职业独立管理，弃誓者可通过立誓重新确立信仰
- **图片输出** — 排行榜支持图片模式，渐变暗色主题，金/银/铜排名高亮
- **批量录入** — 粘贴游戏结算文本自动解析，批量更新积分（支持正负分）
- **群级设置** — 每个群可独立设置输出模式（文本/图片）
- **全局白名单** — 白名单一次添加，所有群生效
- **定时推送** — 每日定时推送排行榜到指定群（支持文本/图片模式）
- **自动备份** — 数据自动备份，可配置保留天数
- **指令可配** — 所有指令名可通过 WebUI 配置修改
- **查询冷却** — 防刷机制，可配置冷却时间
- **直接回复** — 指令回复不经过 AI 处理，固定格式输出

## 指令列表

| 指令 | 别名 | 功能 | 权限 |
|------|------|------|------|
| `天梯榜` | `ladder` `ranking` `排行榜` | 显示登神之路排行榜 | 所有人 |
| `觐见榜` | `pilgrimage` `觐见` | 显示觐见之梯排行榜 | 所有人 |
| `查询 <玩家名>` | `query` `查看` | 查询指定玩家信息 | 所有人（有冷却） |
| `录入玩家 <姓名> <信仰> <职业> [分数]` | `register` `添加玩家` | 录入新玩家（分数可省略，使用默认值） | 白名单/管理员 |
| `设置职业 <玩家名> <职业>` | `setclass` `改职业` | 修改玩家职业（仅职业） | 白名单/管理员 |
| `立誓 <玩家名> <信仰>` | `takeoath` `立约` | 设置玩家信仰 | 白名单/管理员 |
| `弃誓 <玩家名> [新信仰]` | `abandoath` | 标记弃誓者，可选同时改信仰 | 白名单/管理员 |
| `录入积分 <玩家名> <登神之路变化> <觐见变化>` | `addscore` `加分` | 录入积分变化 | 白名单/管理员 |
| `批量录入 <结算文本>` | `batch` `bl` | 粘贴结算文本批量录入积分 | 白名单/管理员 |
| `输出模式 <text/image>` | `outputmode` `模式切换` | 切换本群输出模式 | 管理员 |
| `白名单 <add/remove/list>` | `whitelist` `wl` | 管理全局白名单 | 管理员 |
| `天梯榜管理 <操作>` | `ladderadmin` `榜管理` | 管理员操作 | 管理员 |
| `天梯榜帮助` | `ladderhelp` `帮助` | 显示帮助信息 | 所有人 |

## 权限说明

| 级别 | 说明 |
|------|------|
| **所有人** | 任何群成员均可使用 |
| **白名单/管理员** | 需在 WebUI 白名单配置或通过 `/白名单 add` 添加，管理员不受此限制 |
| **管理员** | 需在 WebUI 的 `admin_ids` 中配置，或为 AstrBot 群管理员/群主 |

### 弃誓文本默认值

`{name}` 会被替换为玩家名：

| 信仰 | 默认文本 |
|------|----------|
| 虚无 | `{name}背弃了虚无之道。万法归寂，誓约已碎。` |
| 存在 | `{name}否认了存在之意。所信皆空，誓约已碎。` |
| 文明 | `{name}舍弃了文明之光。秩序崩塌，誓约已碎。` |
| 沉沦 | `{name}离开了沉沦之渊。深渊不再接纳，誓约已碎。` |
| 混沌 | `{name}背离了混沌之源。混乱不再眷顾，誓约已碎。` |
| 生命 | `{name}抛弃了生命之根。生机断绝，誓约已碎。` |

## 数据存储

- **数据库** — SQLite，位于 `AstrBot/data/astrbot_plugin_faith_ladder/ladder.db`
- **配置** — AstrBot 自动管理，位于 `AstrBot/data/config/`
- **备份** — 位于 `AstrBot/data/astrbot_plugin_faith_ladder/backups/`

### 数据库表

| 表名 | 说明 |
|------|------|
| `players` | 玩家信息（ID、群号、名字、职业、信仰、双积分、弃誓标记） |
| `score_history` | 积分变更历史记录 |
| `whitelist` | 全局白名单 |
| `active_groups` | 活跃群记录 |
| `group_settings` | 群级设置（如输出模式） |

## 安装

1. 将 `astrbot_plugin_faith_ladder` 目录上传到 AstrBot 的 `data/plugins/` 目录
2. 在 AstrBot WebUI 中启用插件
3. 在 WebUI 配置中设置管理员 ID 和白名单
